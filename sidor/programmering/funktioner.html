Funktioner

Vad är en funktion?

Funktioner används för att återanvända samma kod på flera ställen i ett program. Funktioner gör det möjligt att skriva modulär kod vilket gör att man kan dela upp sin kod i mindre mer läsbara delar. Det är bättre att skriva flera små funktioner än enstaka stora funktioner. Funktioner kan ta emot parametrar som indata för att sedan bearbete de och retunera ett resultat

Python

När man vill skapa en funktion i python så skriver man först def och sedan vad man vill kalla den, paranteser och sist ett kolon.

def scream():
    print("aaaaaa!")

Koden man vill att funktionen ska köra skriver man inuti funktionen genom att indentera den. Klicka på tab↹ knappen för att indentera

Vill man mata in parameterar till funktionen så skriver man vad man vill kalla de parametrarna inuti paranteserna med ett kommatecken mellan varje parameter.

def addition(x, y):
    print(x + y)

addition(10, 5)

I funktionen ovan så matar man in 2st tal som funktionen sedan adderar och printar ut. För att köra funktionen så skriver man namnet på funktionen följt av paranteser och om funktionen tar emot parametrar så skriver man de mellan paranteserna.

Vill man att funktionen ska returnera något alltså svara med ett resultat eller en status så skriver man return och sedan vad den ska retunera

def addition(x, y):
    return x + y

result = addition(10, 5)
print(result)

I det här fallet så kallas addition och man matar in 10 å 5 som parameterar, funktionen adderar sedan ihop de och retunerar tillbaka svaret (15) vilket sparas i result variabeln.

Javascript

När man vill skapa en funktion i Javascript så skriver man först function och sedan vad man vill kalla den, paranteser och sist curly brackets

def scream(){
    console.log("aaaaaa!")
}

Koden man vill att funktionen ska köra skriver man mellan curly bracketsen.

Vill man mata in parameterar till funktionen så skriver man vad man vill kalla de parametrarna inuti paranteserna med ett kommatecken mellan varje parameter.

function addition(x, y){
    console.log(x + y)
}

addition(10, 5)

I funktionen ovan så matar man in 2st tal som funktionen sedan adderar och printar ut. För att köra funktionen så skriver man namnet på funktionen följt av paranteser och om funktionen tar emot parametrar så skriver man de mellan paranteserna.

Vill man att funktionen ska returnera något alltså svara med ett resultat eller en status så skriver man return och sedan vad den ska retunera

function addition(x, y){
    return x + y
}

result = addition(10, 5)
console.log(result)

I det här fallet så kallas addition och man matar in 10 å 5 som parameterar, funktionen adderar sedan ihop de och retunerar tillbaka svaret (15) vilket sparas i result variabeln.
