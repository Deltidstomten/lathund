Loopar

Runt och runt och runt och runt...

Loopar används när man vill att en viss kod ska upprepas ett x antal gånger eller tills något kriterium har möts. Det finns 3 sorters loopar, for loop, for each loop och while loop. for och for each loopar loopar ett x antal gånger medan while loopar följer ett kriterium är True.

Python

Python är speciellt för där finns inte vanliga for loopar, bara for each loopar och while loopar.

fruits = ["äpple", "apelsin", "melon", "kiwi"]
for x in fruits:
    print(x)

Programmet ovan går igenom alla element i fruits listan och printar de. X är namnet som varje element i listan kallas i for loopen, det går att ändra och rekomenderas att namge det något mer passande som fruit i det här sammanhanget.

for i in range(10):
    doStuff()

Det går att göra vanliga for loops i python genom att använda range(), den här loopen kommer att loopa 10st gånger och i blir vilket steg / iteration den är på dvs första gånger den loopar är i=0, andra gången i=1, tredje gången i=2 osv. 

while True == True:
    print("om och om")

While loopar kör på sålänge kriteriumet är lika med True, i det här fallet så är True == True så den loopen kommer att köra i all evighet.

Javascript

For loops ser lite läskigare ut i Javascript men det är dom inte.

for (let i = 0; i < 10; i++) {
    console.log(i)
}

I Javascript så måste man definiera ett start nummer, i = 0;. Sedan måste man definera hur länge den ska loopa, i < 10; och sist hur mycket den ska öka med i slutet av varje iteration; i++ vilket bara betyder +1. Det gör att den här for loopen loopar 10st gånger.

while (true == true) {
    console.log("om och om")
}

While loopar kör på sålänge kriteriumet är lika med true, i det här fallet så är true == true så den loopen kommer att köra i all evighet.
