Vad är programmering?

Programmering går ut på att skriva kod som styr en dator, telefon, robot mm. Alla datorer har en processor som följer fördefinerade instruktioner. Allt programmering handlar om är att använda sig av dessa instrukioner i olika ordningar så att datorn gör det vi vill.

Python

I programmering 1 så använder vi oss av programmeringsspråket Python. Python är ett highlevel objektorienterat språk som är ett bra nybörjarspråk. Nedan är ett exempel på hur python kod kan se ut.

id = "8112189876"
numbers = ""
number_total = 0
control_number = 0

for i in range(len(id) - 1):
    if(i % 2 == 0):
        numbers += str(int(id[i]) * 2)
    else:
        numbers += id[i]

for num in numbers:
    number_total += int(num)

control_number = (10 -(number_total % 10)) % 10
