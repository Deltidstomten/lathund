If Satser

Om när gör det

Om man vill att ett program ska göra olika saker beroende på vad användaren har valt eller gjort så används if satser. Programmet kan därför ta olika beslut beroende på vilka krav som förfylls. Nedanför hittar ni flera exempel på hur if satser kan se ut.

Python

if x > 10:
    print("X är större än 10")


if password == "vinter2012":
    print("Välkommen in!")
else:
    print("Fel lösenord!")

if month == "April" and day == 4:
    print("Grattis på din födelsedag!")
else:
    print("Det är inte din födelsedag idag...")

Javascript

if (x > 10){
    console.log("X är större än 10")
}
                
if (password == "vinter2012"){
    console.log("Välkommen in!")
} else {
    console.log("Fel lösenord!")
}

if (month == "April" && day == 4){
    console.log("Grattis på din födelsedag!")
} else {
    console.log("Det är inte din födelsedag idag...")
}
