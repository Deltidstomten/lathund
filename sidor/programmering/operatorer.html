Operatörer

Vad är en operatör?

En operatör i programmering är en symbol eller ett tecken som utför en specifik operation på variablar eller annat nummer. I programmering så kategoriserar operatörerna i följande kategorier:

Aritmetiska Operatörer

Aritmetiska operatörer används för att räkna matte.

Operatör Namn Exempel
+ Addition x + y
- Subtraktion x - y
* Multiplikation x * y
/ Division x / y
% Modulus (rest) x % y

Jämförelseoperatörer

Operatör Namn Exempel
== Lika med x == y
!= Inte lika med x != y
> Större än x > y
< Mindre än x < y
>= Större än eller lika med x >= y
<= Mindre än eller lika med x <= y

Logiska Operatörer

Python JavaScript Namn
and && Logiskt och
or || Logiskt eller
not ! Logiskt inte
