Typer

Olika datatyper

Typer bestämmer vilken sorts data en variabel lagrar, vilket hjälper datorn att hantera den korrekt. Utan typer kan datorn inte skilja mellan siffror och text, t.ex. om 1 + 1 ska bli 2 eller "11". Variabler och konstanter får endast bestå av 1 typ.

Python

I python så tilldelas typer automatiskt, man behåller alltså inte skriva vilken typ en variabel har. De vanligaste typerna i python är

Vanliga typer

Typ Exempel Användning
String "Hej", "Hassan" Texter, namn
Integer 10, 555, 1337 Heltal
Float 10.25, 3.14, 5332.23 Decimaltal
Boolean True, False Sant eller falskt
None None Inget värde.

Advancerade typer

Typ Exempel Användning
List [1,2,3,4], ["banan", "melon", "citron"] Listor med variabler inuti
Dictionary {"namn" : "Hassan", "ålder" : 37, "kön" : "Man"}
{"apple" : "äpple", "orange" : "apelsin", "carrot" : "morot"} 		Koppla samman olika koncept eller översätta
