Variablar & Konstanter

Hur kommer program ihåg?

För att kunna låsa upp din telefon så behöver man mata in en pinkod, hur kommer telefonen ihåg vad din pinkod är?

Variablar gör det möjligt för program att spara information i datorns minne, tänk att datorn har en stor samling av luckor med olika namn, inuti de luckorna så kan man förvara information, det är vad en variabel är. En variabel kan hålla flera olika sorters data med helt olika storlekar, allt ifrån vilken dag det är idag till alla namn på alla elever på skolan.

När det finns information som man inte vill ska gå att ändra på som hur många dagar det finns varje vecka eller hur PI ser ut. Då kan man istället spara informationen i en konstant. En konstant liknar en variabel fast så fort man har gett den information så går det inte att ändra den.

Python

När man vill skapa en variabel i python så skriver man namnet på variabeln, ett likamedtecken och sedan vad den ska innehålla.

pinkod = 1234
namn = "Hassan"
stad = "Östersund"
postkod = 83253

När en variabel består av flera ord så ersätter man varje mellanslag med ett understreck

väldigt_långt_namn = 23

Variabler ska ALLTID börja med småbokstav och de får inte börja med något tecken som inte är en vanlig bokstav.

Det finns inte konstanter i python

Javascript

När man vill skapa en variabel i javascript så skriver man let, namnet på variabeln, ett likamedtecken och sedan vad den ska innehålla.

let pinkod = 1234
let namn = "Hassan"
let stad = "Östersund"
let postkod = 83253

När en variabel består av flera ord så tar man bort varje mellanslag och gör den första bokstaven i ordet till storbokstav

väldigtLångtNamn = 23

Variabler ska ALLTID börja med småbokstav och de får inte börja med något tecken som inte är en vanlig bokstav.

För att göra en variabel till en konstant så byter man ut let mot const samt gör namnet till stora bokstäver

const PI = 3.141592653589793
const NAMN = "Hassan"

När en konstant består av flera ord så ersätter man varje mellanslag med ett understreck

const DAGAR_I_VECKAN = 7
