Element / Taggar

Intro

HTML är ett språk som man använder när man vill skapa hemsidor. HTML är uppbyggt med taggar som ser ut såhär <tag> text </tag>

Genom att placera ut flera av dessa taggar så kan man bygga upp en hemsida

Nedan är ett exempel på en hemsidas kod

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>

    <h1>My First Heading</h1>
    <p>My first paragraph.</p>

</body>
</html>

(ni kan visa den här hemsidans kod genom att högerklicka och välja "visa källkod")

Alla hemsidor börjar med DOCTYPE taggen som förklarar för webbläsaren att det här är en hemsida. Efter det kommer html taggen som säger att allting mellan taggen är html kod

Sedan har vi Head och Body taggarna. I head lägger vi alla taggar som ger extra information till webbläsaren som bl.a vad titlen ska vara, vilken text kodning som ska användas och om det finns andra filer som den ska hämta information från. I body så lägger vi alla taggar som påverkar utseendet på hemsidan som bl.a text, bilder, länkar och listor

Det här är grunden till en hemsida, genom att fylla på med flera taggar så kan man sakta men säkert bygga upp en fungerande hemsida.

Attribut

Alla taggar kan man tilldela attribut som ger extra information om taggen.

Exempel: <img src="bild.jpg"> img taggen visar en bild. src attributet gör så att webbläsaren vet vart den ska leta efter bilden.

Exempel: <p class="red-text"></p>class attributet i p taggen säger till webbläsaren att följa den namngivna klassens css regler.
