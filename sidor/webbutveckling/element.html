Element / Taggar

HTML är ett språk som man använder när man vill skapa hemsidor. HTML är uppbyggt med taggar som ser ut såhär <tag> text </tag>

Genom att placera ut massor av dessa taggar så kan man bygga upp en hemsida

Nedan är ett exempel på en hemsidas kod

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>

    <h1>My First Heading</h1>
    <p>My first paragraph.</p>

</body>
</html>
