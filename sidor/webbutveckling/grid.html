Grid

Intro

Grid används för att enkelt placera ut element i ett rutnät med hjälp av rader & kolumner. Grid är väldigt hjälpsam om man vill göra responsiva sidor.

Egenskaper

För att använda grid så måste man minst definera hur många kolumner man vill att rutnätet ska bestå av.

Med hjälp av grid-template-columns och grid-template-rows så kan man definera hur rutnätet ska se ut.

Exempel

.grid {
    display: grid;
    grid-template-columns: auto auto;
    grid-template-rows: auto auto;
}

<div class="grid">
    <button>1</button>
    <button>2</button>
    <button>3</button>
    <button>4</button>
</div>
