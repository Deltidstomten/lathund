CSS

Intro

CSS är ett språk som används för att ändra hemsidors utseende. Med CSS så kan man skapa en stilmal där man definerar regler över hur saker ska se ut. Det finns väldigt många olika regler man kan ändra, här är en länk till ett index av css regler. CSS Reference Index

Syntax

CSS är inte uppbyggd med taggar som HTML. Istället så skapar man regler som ser ut sähär:

body {
    background-color: red;
    font-size: 20px;
}

I början så har vi en selector som beskriver vad som ska följa den här regeln, i det här fallet så står det body vilket betyder att allting inuti body följer denna regel. Det följs av ett par Curly brackets vilket skämtsamt brukar kallas för måsvingar på svenska. Mellan måsvingarna så skriver man de attributen som ska ändras. i det här fallet så ska bakgrundsfärgen bli rött och all text ska vara 20 pixlar stor.

Selector

Det finns flera olika sätt att bestämma vilka taggar som ska påverkas av specifika regler. De 2 vanligaste sätten är med en typ selector eller en class selector.

Typ selectorn så skriver man namnet på taggen som ska påverkas såhär:

p {
    color: red;
}

h2 {
    color: blue;
}

De här 2 reglerna gör så att alla p taggar har röd text och att alla h2 taggar har blå text.

Sedan har vi class selector vilket är när man skapar en class som man sedan delar ut i HTML koden till de taggar man vill ska följa den regeln.

Såhär hade den koden sätt ut.

.red-text {
    color: red;
}

.blue-text {
    color: blue;
}

Först skriver man en punkt för att visa att det är en class och sedan det man vill kalla classen

När man vill dela ut classen till specifika taggar så gör man såhär:

<h1 class="red-text">My First Heading</h1>
<p class="blue-text">My first paragraph.</p>

Man delar alltså ut class attributen till taggarna där man skriver namnet på classen.
