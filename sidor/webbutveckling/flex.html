Flex

Intro

Flex används för att enkelt placera ut element i antligen en kolumn eller en rad. Flex är väldigt hjälpsam om man vill göra responsiva sidor. Det är väldigt vanligt att använda flex när man vill skapa en navbar

Egenskaper

I Flex finns två axlar: huvudaxeln (x-axeln) och korsaxeln (y-axeln). Med flex-direction kan man ändra riktningen så x och y axeln byterplats eller inverteras.

Exempel

.flex{
    display:flex;
    align-items: center;
    justify-content: space-between;
}

.flex-item{
    background-color: #283955;
    color: white;
    padding: 10px;
    border-radius: 20px;
    width: fit-content;
}

<div class="flex">
    <p class="flex-item">Text1</p>
    <p class="flex-item">Text2</p>
    <p class="flex-item">Text3</p>
    <p class="flex-item">Text4</p>
    <p class="flex-item">Text5</p>
</div>
