Git & Github

Intro

Git är ett versionshanteringprogram som används när man spara, dela och hålla koll på kod.

Git använder vi för att både säkerhetskopiera eran kod samt för att dela eran kod med läraren.

Github är sidan som vi pekar git mot och blir därför platsen där koden kopieras till.

Hur man sätter upp git

  1. Skapa ett repository på github
  2. Öppna terminal / cmd i mappen som git ska hantera
  3. skriv > git init för att initialisera git
  4. skriv > git branch -M main för att sätta branchen till main vilket är standarden på github.
  5. skriv > git remote add origin länk_till_er_repository.git för att konfigurera vart git ska kopiera
  6. Klar!

Nu har ni satt upp git och kan börja använda programmet.

Hur man använder git

  1. Öppna terminal / cmd i mappen som git hanterar
  2. skriv > git add . för att markera vilka filer som ska kopieras, punkt betyder alla filer.
  3. skriv > git commit -m "kommentar" för att förberada en commit dvs en överlämning av filer.
  4. skriv > git push origin main för att pusha, vilket skickar iväg filerna till github.
  5. Klar!

Hur man kommer åt sina filer på en annan dator

Om man vill jobba på en annan dator så är det viktigt att följa några annorlunda steg för att inte riskera att förstöra sitt repository på github.

  1. Öppna terminal / cmd i mappen som git ska hantera
  2. skriv > git init för att initialisera git
  3. skriv > git branch -M main för att sätta branchen till main vilket är standarden på github.
  4. skriv > git remote add origin länk_till_er_repository.git för att konfigurera vart git ska kopiera
  5. skriv > git pull origin main för att pulla, vilket laddar hem alla filer från github.
  6. Klar!

    7. OBS! Det är viktigt att alltid pusha sin kod när man är klar och sedan pulla koden om man byter dator för att inte skapa konflikter i sitt repository
